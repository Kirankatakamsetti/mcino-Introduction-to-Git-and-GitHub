# Simple Interest Calculator

A simple interest calculator script written in Bash.

## Description
This script calculates simple interest given the principal, rate of interest, and time period.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- Principal amount
- Rate of interest (per annum)
- Time period (in years)

## Output
- Simple Interest
